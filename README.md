Heroku buildpack: jq
=======================

This is a [Heroku buildpack](http://devcenter.heroku.com/articles/buildpacks) of [jq](https://stedolan.github.io/jq/).

Usage
-----

Example usage:

    [your app]
    $ heroku buildpacks:add 1 http://github.com/faircapital/heroku-buildpack-jq.git
